# 🧠 Brain Tumor Detection using CNN (Scratch)

## 📌 Project Overview
This project uses a Convolutional Neural Network (CNN) built from scratch to classify brain MRI images into 4 categories:
- Glioma
- Meningioma
- No Tumor
- Pituitary

## 🚀 Technologies Used
- Python
- PyTorch
- Torchvision
- Matplotlib

## 📊 Dataset
MRI dataset containing 4 classes with training and testing images.

## 🧠 Model Architecture
- 2 Convolutional Layers
- ReLU Activation
- MaxPooling
- Fully Connected Layers

## 🎯 Results
Achieved good accuracy on test dataset using CNN from scratch.

## 📂 How to Run
1. Clone repo
2. Open CNN_Scratch.ipynb
3. Run all cells
4. Train model
